# Application Update Engine
 
Updater [/F] /E:source [/L[[:logfile/logext]]] [/X:[.]tmp] [/T:path]
 
- /? : Display help
- /F : Replace existing files
- /L : Log operations, [log file or log extension]
- /E : Update engine
- /X : Cache file extension
- /T : Target directory
- /I : Get available plug-ins
